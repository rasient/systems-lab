# 🌍 Systems Lab — Exploring How Systems Actually Work

This is not a collection of separate projects.

It’s a set of experiments.

---

## 🧠 Core Idea

Systems don’t fail because of components.

They fail because of:

* coordination
* interfaces
* misaligned incentives

---

## 🔍 What This Represents

A set of prototypes exploring:

→ infrastructure systems
→ energy systems
→ water systems
→ AI content systems
→ decision & routing systems
